# fbcmd4j

## Installation
```
git clone https://github.com/ricardo2728814/fbcmd4j.git
```
## Usage
Open the jar file:
```
cd .\dist\
java -jar .\fbcmd4j.jar
```
Type any of the following commands:
- feed
- usuario
- salir

Follow the instructions on the console.
Enjoy!
